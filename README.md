Project Description
This project presents a smart classroom automation system based on IoT concepts using the ESP32 microcontroller. The system is designed to automatically control classroom lighting and fan operation based on human presence and surrounding light conditions. The automation improves energy efficiency and reduces manual intervention in electrical appliance control.

Objective
The objective of this project is to develop an intelligent automation system that minimizes power wastage in classrooms by automatically operating lights and fan using motion detection and ambient light sensing.

Components Used
ESP32 Development Board
PIR Motion Sensor
LDR (Light Dependent Resistor)
Relay Module
LED / Bulb
DC Fan
Connecting Wires
Power Supply

Working Principle
The PIR motion sensor continuously monitors human movement inside the classroom. When a person is detected, the ESP32 activates the DC fan through the relay module to provide ventilation.
Simultaneously, the LDR sensor measures ambient light intensity. If the surrounding environment is dark, the lighting system is automatically switched ON.
When no human presence is detected for a predefined time interval, both the fan and lighting system are automatically turned OFF to conserve electrical energy.

Features of the System
Automatic presence-based fan control
Intelligent lighting based on ambient light level
Energy efficient classroom automation
Low-cost embedded IoT solution
Reduced manual operation

Applications
Smart classrooms
Office room automation
Library and laboratory energy management
Smart building automation systems

Future Scope

Conclusion
The proposed system demonstrates an effective implementation of embedded automation for improving energy efficiency in educational environments. The project highlights practical use of sensors, microcontrollers, and relay interfacing for developing smart infrastructure solutions.
